# Imágenes Cholao Oscar

Assets del negocio. En la app se referencian como `/images/<nombre>`.

| Archivo | Descripción |
|---------|-------------|
| `logo.JPG` | Logo — base para favicon y branding |
| `cholao.JPG`, `cholao2.JPG` | Fotos del negocio |
| `delivery.JPG`, `cholaoDelivery.JPG` | Delivery |
| `maracuyazo.JPG` | Producto |

## Favicon

1. [favicon.io](https://favicon.io) → subir `logo.JPG`
2. Descargar paquete → copiar a `/public/` y `/public/icons/` (ver `AGENTS.md`)
